Program's name: RECIPE AND MATERIALS(BOOKS) STORAGE

Description: The progam allows user to access recipes or books such as inserting,
             deleting, displaying and searching. The program uses the API sqlite3
             to generate the SQL structure and code in Python.

How to run: the program is a command-line program, so firstly user need to enter command line interface (Linux or Windows cmd),
            then goes the direction which contains files (recipeMain.py and recipe.db).
            After that, user types "python3 recipeMain.py" in command line and press ENTER.
            Then user just need to follow the instructions when the program runs.
